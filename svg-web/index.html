默认一个空白 svg


Chrome Safari 浏览器默认 300*150

和 img 一样，默认情况会高出一些像素


矩形 rect








rect 默认黑色填充。核心属性 width height，没有这两个就不会出现形状。

「fill="none"」透明一样。

stroke 按表现看是在元素边缘内湾各扩散半个宽度的方式渲染。如果要完整显示出来，图像的尺寸应该是「矩形尺寸 + stroke-width / 2」。

x y 是元素起点坐标，为了比较好一像素一像素的渲染，「stroke-width」设置为偶数应该比较好。在两个像素之间画一条线，左右或上线两个像素点只能画半边，边缘虚化。偶数正好占两个像素点。

「stroke-opacity」边缘填充的透明度，0~1。

「fill-opacity」内部填充的透明度，0~1。

「fill」、「stroke」填充颜色是 「rgb」「rgba」「#16进制」。

「*-opacity」和「rgba」两个方式透明表现上没什么差别。

rx ry 可以制作圆角，两个值相同就是很圆的。rx ry 从表现上是按照四个角，分别在 x、y 轴相距 n 像素，做一个曲线。


圆形 circle





circle 默认黑色填充。核心属性 cx cy r，没有这两个就不会出现形状。

「fill="none"」透明一样。

stroke 渲染方式表现和在矩形 rect 中差不多。内外各占/绘制一般。

填充、边缘应该也是一样的表现。

通过 r 来控制 stroke 和整个元素视觉大小。


椭圆 ellipse





ellipse 默认黑色填充。核心属性 cx cy rx ry，没有这几个就不会出现形状。

rx ry 以中小点在 x 轴左右 y 轴上线为边界画一个矩形，按照四个顶点画圆弧、border-radius。

「fill="none"」透明一样。

stroke 渲染方式表现和在矩形 rect 中差不多。内外各占/绘制一般。

填充、边缘应该也是一样的表现。

通过 rx ry 来控制 stroke 和整个元素视觉大小。


直线 line

line 默认宽 1 像素。核心属性 x1 y1 x2 y2 stroke，没有这几个就不会出现形状。

stroke 渲染方式表现和在矩形 rect 中差不多。内外各占/绘制一般。

其他属性也相似。
